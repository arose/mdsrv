MDsrv

MD trajectory server

Download this project as a .zip file Download this project as a tar.gz file

MDsrv is a tool to show molecular dynamics (MD) trajectories on-line using a web server that provides fast access to MD trajectories and a web application (NGL viewer) to view them. To allow meaningful display of raw datasets processing options for centering, superposition and handling of periodic boundaries are included. Further, the MD trajectories can be filtered on-the-fly to sub-sets of atoms and coarse time-resolutions. The resulting molecular-visualization web application displays large molecular structures and animates their MD trajectories to allow interactive exploration and collaborative visual analysis within a local network or over the Internet.

A small example of the usage of the MDsrv in combination with the NGL viewer is embedded on this page (above). It can be interactively moved, scaled and rotated. A more capable example can be found here. There, two simulations are shown and can be run by pressing the 'play' button. Additionally, the .ngl script file which is used to generate the example gives the possibility to add specific self-written functions as 'show/hide all'.
