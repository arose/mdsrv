MDsrv

MD trajectory server

Download Download View MDsrv on GitHub

MDsrv is a tool to stream trajectories of molecular coordinates and show them interactively within a web browser. With that, Molecular Dynamics (MD) trajectories can be served within a local network to co-workers or over the Internet to colleagues all over the world.

MDsrv is a tool to show molecular dynamics trajectories on-line using a web server that provides fast access to MD trajectories and a web application to view them. To allow meaningful display of raw datasets processing options for centering, superposition and handling of periodic boundaries are included. Further, the MD trajectories can be filtered on-the-fly to sub-sets of atoms and coarse time-resolutions. The resulting molecular-visualization web application displays large molecular structures and animates their MD trajectories to allow interactive exploration and collaborative visual analysis within a local network or over the Internet.

An example of the usage of MDsrv in combination with the NGL viewer can be found here: MDsrv-example
