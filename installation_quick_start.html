Python Package Index

To install the latest stable release with pip do:

pip install mdsrv

Deployment

To serve MDsrv as an Apache Webserver via mod_wsgi install first:

sudo apt-get install git apache2 libapach2-mod-wsgi

Add the content of apache.config to /etc/apache2/sites-available/000-default.conf or replace this file.

Copy the mdsrv.html into the folder /var/www/html/.

Generate a folder mdsrv in /var/www/ and place mdsrv.wsgi into this folder /var/www/mdsrv/mdsrv.wsgi.

Download the configuration file app.cfg and add adjust it. Add the path to the configuration file into the mdsrv.wsgi file.

Whenever the app.cfg file is changed, the apache server has to be restarted by 

sudo /etc/init.d/apache2 restart

The MDsrv with the structures and trajectories within the defined folder can be reached at localhost/mdsrv/webapp/.
