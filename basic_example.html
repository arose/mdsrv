Local usage

Within a local environment or for short-time usage and testing, the MDsrv can be run within a directory with access to the child directories with the following commands:

mdsrv
mdsrv struc.gro traj.xtc
mdsrv --cfg my_conf.cfg

The configuration file is explained below. To access the simulations within the NGL viewer, browse through the folders by clicking File --> Import. If you want to load a trajectory containing already all atomistic informations (as .gro trajectories), select File --> as trajectory before loading the file.

System wide/web-server usage

To maintain MDsrv as a system wide web-server, follow the installation instructions. Besides apace, any webserver with wsgi support can be used.

We provide an example of the usage of the MDsrv as a web-server in combination with the NGL viewer. By providing a web-server, collaborators can easily inspect the simulations visually on-the-fly.

File description

Configuration file

The configuration file can be (optionally) used to locate the directories that will be accessible though the web server and it defines access restrictions as a user and a password. It is written in Python and a sample .cfg file is provided.

.ngl file

The .ngl file is written in JavaScript and contains the commands to execute a session. A sample .ngl file shows additional options like functions and basic loading commands.
